ECMAScript 2015; getting our hands dirty

Since you’re reading this, I’m going to make the assumption that you’ve heard a few times about ECMAScript 2015, also known as “ECMAScript 6”, “JavaScript Next”, and probably quite some other more or less fortunate names.

If you’re anything like me, you will be pretty excited to start learning more about the new language version; what changes, the new features, and so on. Funny enough, I’ve come across five different ES6-related resources right this morning, and I thought I might as well share them.

  • Six Steps for Approaching the Next JavaScript: Useful, “first-approach” kind of post.
  • ECMAScript 6 features: Very comprehensive review, with examples, of all the new features that ES6 introduces. Works great as a cheatsheet.
  • Learn ES6 from AirBNB’s coding standards: Valuable recopilation of code examples.
  • Exploring ES6: Very detailed, +460-page, free to read book by Axel Rauschmayer.
  • Lastly, the Babel online compiler, so we all can easily start to play with this new toy…

So, learn a lot and enjoy!
